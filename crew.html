#불러오는 중...
오늘 참여 링크 (해시태그 누락)
해시태그 누락된 크루의 참여 링크 — 클릭해서 좋아요를 눌러주세요
불러오는 중...
오늘 할 일
✅
오늘 참여 완료!
참여 규칙
①참여 시간 10:00~21:59 / 완료 시간 22:00~익일 09:59
②해시태그 복사 후 게시물 댓글에 작성 → 참여 등록
③완료 시간에 해시태그 2~3회 재검색, 꼼꼼히 좋아요
④밴일 경우에만 밴 링크 추가 (허위 밴 봉사 처리)
⑤주 2회 이상 참여 필수
봉사 규칙
①주 0회 → 봉사 2일 / 주 1회 → 봉사 1일 (매주 일요일 자동)
②완료 미등록 → 봉사 1일 자동 부여
③봉사 발생 후 3일 이내 미처리 시 계정 자동 잠금 (운영진 해제 필요)
④해태/무임승차/먹튀 → 연계방 전체 차단 (1,500명+)
입력 기호
●일반 참여 (피드있음) — 주참여수 포함
N노피드 참여 — 주참여수 포함
B봉사, 피드없음 — 주참여수 미포함
F봉사, 피드있음 — 주참여수 포함
📊현황
✋ 참여 등록
1인 1회 참여만 가능합니다. 해시태그에 누락된 경우 아래에서 참여 링크를 등록해주세요.
해시태그에 내 게시물이 안 뜨는 경우 링크를 등록하면 다른 크루에게 공유됩니다
✅ 완료 체크
해시태그 2~3회 재검색 후 꼼꼼히 좋아요 눌렀나요? 밴 링크도 챙겨주세요!
🔗 참여 링크 등록
해시태그 검색에 내 게시물이 안 뜨는 경우에만 등록해주세요.
등록된 링크는 다른 크루에게 공유됩니다.
✈️ 휴가 신청
휴가 기간 동안은 봉사가 면제됩니다. 운영진 승인 후 적용됩니다.
